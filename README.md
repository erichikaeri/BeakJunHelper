# BaekJunHelper
백준에서 예제 입력 가져와서 테스트 해주는 프로그램입니다.

## 지원언어

1. Python

2. C++

## 설치
`Install.bat`

## 실행
`Run.bat`

## 사용법

1000번 문제로 해볼게요 (https://www.acmicpc.net/problem/1000)

#### 2
