# BaekJunHelper
백준에서 예제 입력 가져와서 테스트 해주는 프로그램이므

## 설치
`pip install requests beautifulsoup4`

## 실행
`python Tester.py`

## 사용법

1000번 문제로 해볼게용 (https://www.acmicpc.net/problem/1000)
